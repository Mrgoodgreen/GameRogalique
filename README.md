# XYZRoguelike

**XYZRoguelike** — это 2D-игра, разработанная на языке C++ (стандарт C++17) с использованием мультимедийной библиотеки SFML. Игрок бегает по лабиринту и убивает монстров. Проект построен на базе собственного простого 2D движка **XYZEngine** (построено на компонентной архитектуре).

## Особенности проекта
* **Собственный движок XYZEngine:** включает системы рендеринга, базовой физики, управления ресурсами, звуком и камерой.
* **Игровые механики:** подгрузка уровней, противники с базовым ИИ, система здоровья и урона.
* **Логирование и отладка:** встроенная система логов и макросов утверждений (Assertions).
* **Модульная архитектура:** код игры отделен от библиотек движка.

## Как запустить игру

### Способ 1: Сборка и запуск через Visual Studio (рекомендуется)
1. Откройте файл решения `Game.sln` в **Visual Studio**.
2. Убедитесь, что выбрана конфигурация **Release** или **Debug** для платформы **x64**.
3. Сделайте проект `XYZRoguelike` запускаемым по умолчанию (правый клик по проекту -> *"Сделать запускаемым проектом"*).
4. Нажмите `F5` (или `Ctrl + F5`), чтобы собрать и запустить игру. При таком запуске рабочая директория будет настроена автоматически, и игра найдет все ресурсы.

### Способ 2: Запуск скомпилированной версии (.exe)
Если вы хотите запускать игру напрямую из папки через собранный `.exe` файл, вам необходимо убедиться, что игра может найти свои ресурсы:
1. Перейдите в папку с собранным `.exe` файлом (`x64\Release\` или `x64\Debug\`).
2. **Обязательно скопируйте** папку `Resources` из директории `XYZRoguelike/Resources` в папку с исполняемым файлом (чтобы она находилась рядом с `XYZRoguelike.exe`).
3. Убедитесь, что библиотека `openal32.dll` (от SFML Audio) лежит в этой же директории (если копируете билд на другой ПК, могут потребоваться и другие .dll от SFML).
4. Запустите `XYZRoguelike.exe`.

> *Примечание: Если игра мгновенно закрывается, скорее всего, она не может найти графические или звуковые ресурсы в папке `Resources`.*

## Как играть

### Управление:
* **W, A, S, D** — перемещение персонажа (вверх, влево, вниз, вправо).
* **Пробел (Space)** — атака (атакует ближайших противников в радиусе действия).
* **F9** — тестовая кнопка для разработчиков (создает намеренный сбой для проверки системы Crash-репортов и логирования).

Продвигайтесь по уровням, избегайте столкновений с врагами и побеждайте их с помощью атак! Расстояние для атаки и кулдаун ограничены.
